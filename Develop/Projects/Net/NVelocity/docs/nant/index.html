NAnt Tasks

NAnt is a build tool built specifically for .Net applications and like NVelocity is based on work done by a Java open source proejct. It offers the ability to extend its functionality by the definition of new tasks.

The following tasks were Ant tasks ported from Velocity and are now availbe as NAnt tasks:

  • Anakia : XML->doc tool   [Velocity site documentation]
  • Texen : text generation   [Velocity site documentation]
  • DVSL : XML xformation   [Velocity site documentation]
